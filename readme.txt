== Installation ==

1. Install plugin and activate it on the Plugins page
2. Go to the "WPML page order" tab under Pages and specify your desired order for pages
4. Modify your template to use correct menu sort order: [wp_list_pages](http://codex.wordpress.org/Template_Tags/wp_list_pages)('sort_column=menu_order&title_li=');
